Web Speech API - Accuracy Test

The purpose of this test is to find out how accurate certain phrases can be detected by the Web Speech API. The test is conducted by . If you have questions or want to give feedback, please write an email to .

Please not that an anonymised report of your test will automatically be sent to the conductor!

Instructions

Press the "Start Test" button and start reading the words and sentences that appear. Speech recognition can take a few seconds. If the phrase was understood correctly, it will automatically change to the next phrase. If the phrase was not correct, the recognized phrase will be displayed. You can then try again. If it does not work repeatedly, you can choose to skip this phrase or select "Nearly correct" if it has at least recognized a similar phrase.

Please try to do the test as fast as possible, but keep in mind that it works best if you talk clearly and not too fast.

After you pressed the "Start Test" button, you will have to allow microphone access. Usually there will be a dialog toolbar under your URL-bar! The test will start immedeatly after you allowed microphone access.
